# DT Fellowship Assignment – Evening Reset Reflection Tree

A deterministic end-of-day reflection tool that guides employees through a structured conversation using fixed choices and predictable branching paths.

---

## Objective

This project was designed for the DT Fellowship assignment to demonstrate how psychological insights can be converted into a deterministic tree system.

The tool helps an employee reflect on their day without using any LLM or free-text input at runtime.

Same answers always lead to the same path and same reflection.

---

## Three Psychological Axes

### 1. Agency (Victim ↔ Victor)
Did I focus on what I could influence, or only what happened to me?

### 2. Contribution (Entitlement ↔ Contribution)
Did I think more about what I gave, or what I deserved?

### 3. Impact Radius (Self ↔ Others)
Did I think only about myself, or also about teammates/customers?

---

## Why Deterministic?

Reflection systems should be:

- Predictable  
- Auditable  
- Trustworthy  
- Consistent  

This project encodes reflection logic into a tree rather than relying on AI-generated responses.
