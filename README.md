# Evening Reset Reflection Tree

A deterministic end-of-day employee reflection tool.

## Purpose
This tool helps employees reflect on their day through a guided tree with fixed choices.

## Core Idea
No AI is used at runtime.  
Same answers always produce the same path and same reflections.

## Three Psychological Axes

1. **Agency**  
Did I respond with control or helplessness?

2. **Contribution**  
Did I focus on giving or receiving?

3. **Impact Radius**  
Did I think only about myself or others too?

## Structure

- Start node
- Question nodes
- Reflection nodes
- Bridge nodes
- Summary node
- End node

## Why Deterministic?

Because reflection tools should be:

- predictable
- auditable
- trustworthy
- consistent

## Files

- `tree/reflection-tree.json` → tree data
